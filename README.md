# 💰 AI Smart Personal Finance Advisor

An intelligent **AI-based Personal Finance Advisor** built using Python, Machine Learning, and Streamlit.
This project helps users analyze their monthly finances, visualize spending patterns, and get smart investment recommendations.

---

## 🚀 Features

### 📊 Financial Analysis

* Calculates total expenses, savings, and savings rate
* Provides detailed category-wise expense breakdown
* Displays percentage of income spent in each category

### 📈 Data Visualization

* Expense Breakdown (Bar Chart)
* Expense Distribution (Pie Chart)
* Income vs Expense vs Savings Comparison
* Stock Allocation Visualization
* Spending Trend & Prediction Graph

### 🤖 AI-Based Suggestions

* Smart financial advice based on spending habits
* Alerts for overspending
* Personalized recommendations to improve savings

### 💼 Investment Recommendations

* Suggests allocation of savings into:

  * Mutual Funds
  * Fixed Deposits
  * Stocks
  * Emergency Funds

### 📈 Stock Portfolio Guidance

* Categorized into:

  * Large Cap (Low Risk)
  * Mid Cap (Medium Risk)
  * Small Cap (High Risk)
* Suggests diversified stock options

### 🔮 Machine Learning Prediction

* Uses **Linear Regression** to predict future expenses
* Shows expected increase/decrease in spending

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/ai-finance-advisor.git
cd ai-finance-advisor
```

### 2️⃣ Install Dependencies

```bash
pip install pandas numpy matplotlib scikit-learn streamlit yfinance
```

---

## ▶️ How to Run

### 🔹 Run Streamlit App (Recommended)

```bash
streamlit run main.py
```

Then open in browser:

```
http://localhost:8501
```

---

### 🔹 Run Console Version

```bash
python finance.py
```
